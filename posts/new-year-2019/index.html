New Year 2019

Jan 2, 2019 · 2 min read

2018 has been a good year, both from a professional and a personal stand-point. But there is always room for improvement, so this is what I hope to achieve in 2019:

  • Read at least one book per month. This one shouldn’t be too hard. I think I read around 7-8 books in 2018, though it’s hard to tell because I haven’t been keeping track. So an additional goal for 2019 is also to track the books that I read throughout the year.

  • Sleep 7-8 hours a day. I mostly do this already, but I need to get better at going to bed earlier.

  • Launch Comment Tracker. I’ve been working on it for maybe a couple of months now, and it’s almost ready for launch. Unfortunately, I don’t know if there is a market for it. There is only one way to find out!

  • Learn something new every month (and write about it). This is as much about learning as it is about getting better at communicating my ideas.

  • Finish my master’s degree. Just two semesters left, so this shouldn’t be too hard!

  • Learn German. Read one German book. I’ll be moving to Berlin, so I really want/need to learn some German.

  • Manage my time better (stop wasting so much time on Reddit/Hacker News). This is the most important (and maybe the most difficult) goal for 2019. If I nail this one, then everything else will fall into place.
